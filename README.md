# Lusail-Stadium-Management-System
Overview

This project is a full system analysis and design for a modern, automated stadium monitoring system inspired by Lusail Stadium in Qatar. It includes requirement gathering, system modelling, database prototyping, and ethical/legal analysis following software engineering industry standards.

Key Features

Requirements engineering (functional + non-functional)

LSEPI analysis (Legal, Social, Ethical & Professional Issues)

5Ps framework applied to real-world system

UML diagrams (use case, sequence, class)

Data Flow Diagrams (DFD)

Prototype relational database design

SQL schema planning

Risk management, system evaluation, and stakeholder analysis

Skills Demonstrated

Software engineering methodology (Scrum vs V-Model)

System modelling & architectural thinking

Technical documentation and academic referencing

Understanding of accessibility, ethics, and data protection

Collaboration within a multi-member development team
